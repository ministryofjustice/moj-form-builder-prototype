Form Builder prototype

The Form Builder platform lets users create, edit and publish forms using components and patterns from the GOV.UK Design System. This prototype has been created in response to the road map session we had in August after the team's focus to create the CFF service.

Design hypotheses

The following overarching hypotheses are useful to understand the reasons for redesigning the Form Builder platform:
